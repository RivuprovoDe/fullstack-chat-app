# 💬 Full-Stack Real-Time Chat Application

This is a full-stack real-time chat application developed as a **learning and portfolio project** to strengthen full-stack web development skills. The application allows users to securely sign up, log in, and communicate with other users in real time.

The project focuses on building a modern frontend, a secure backend, and real-time communication using industry-standard tools and practices.

---

## 🚀 Features

- Secure user authentication using **JSON Web Tokens (JWT)**
- Real-time one-to-one messaging powered by **Socket.IO**
- User profile management with avatar upload support
- Online and offline user status indication
- Responsive and modern UI built with **Tailwind CSS**
- Fast and optimized frontend using **Vite**
- Clean and modular backend architecture

---

## 🛠️ Technologies Used

### Frontend
- React
- Vite
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB & Mongoose
- Socket.IO
- JWT Authentication
